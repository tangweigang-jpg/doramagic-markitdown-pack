load MarkItDown conversion context, file-format boundaries, MCP usage notes, and verification checks into your AI host before turning documents into Markdown

## Two-Minute Path

1. Read `AGENTS.md`.
2. Copy the relevant host instruction into your AI coding host.
3. Run `06_EVALS/smoke_check.md`.
4. If the check fails, open `03_PITFALL_LOG.md`.
